My Zero Shell config files !

Have a quick overview
---------------------
![A screenshot of what my .zshrc does](zsh.png)

Use me
------
Clone me, then

* cd zshrc
* ln -s ./zshrc $HOME/.zshrc       
* ln -s ./zsh/ $HOME/.zsh/ 
* zsh

Include
-------

  * syntax highlight
  * battery state
  * 256 colors
  * various stuff
